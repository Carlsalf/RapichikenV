# RapichikenV (Backend)

Backend del proyecto RapichikenV.

## Repositorios
- Backend (este repo): https://github.com/Carlsalf/RapichikenV
- Frontend: https://github.com/Carlsalf/Carlsalf-FrontSGAVPP

## Carpeta principal
- `PROYECTO_RAPICHIKEN/backSGAVPP/`

## Guía rápida (para ejecutar en otra PC)
1. Clonar el repositorio
2. Entrar a `PROYECTO_RAPICHIKEN/backSGAVPP`
3. Instalar dependencias (`yarn install` o `npm install`)
4. Ejecutar en desarrollo (`yarn develop` o `npm run develop`)
